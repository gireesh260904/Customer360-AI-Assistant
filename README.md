# Customer360 AI Assistant

An AI-powered dashboard that consolidates customer information from CRM, Support, Emails, Slack, and Product Usage into a unified customer view.

## Features

- Unified Customer Dashboard
- AI Customer Summary
- Risk Identification
- Opportunity Detection
- Next Best Action
- Customer Health Score
- AI Chat Assistant

## Tech Stack

- Python
- Streamlit
- LangChain
- Ollama
- Pandas

## Run

pip install -r requirements.txt

streamlit run app.py
